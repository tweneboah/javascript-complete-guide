Blog - official documentation

Blog Settings: Customize > Shopify menu dropdown > Blog > Edit section

Create Article Field Content

  • Copy Article HTML code:
    GET HTML Standart
    GET HTML Gallery
    GET HTML Link
    GET HTML Quote
    GET HTML Video
    GET HTML Audio
  • Paste the code in the "Text Editor" you use. Edit HTML to your needs.
  • In the "Content" field click the "Show Html" button.
  • Copy the code from the "Text Editor" and paste code in the "Content" field
  • Save page

Short blog description Field Excerpt

Overview

  • Short article description. It works in the galleries blog, sidebar blog on the sliders blog.
  • Featured image is the main blog image by default. It works only with the blog thumbnails
  • Some examples of the shortcode description. For the Thumbnail blog in the EXCERPT field
    GET HTML Link
    GET HTML Quote
    GET HTML Audio

Edit field "Excerpt"

  • Copy the HTML code above or create your HTML code in the text editor
  • Edit HTML to your needs.
  • In the "Excerpt" field, click the"Show HTML" button.
  • Copy code from "Text Editor" and paste code in the "Excerpt" field
  • Save page
